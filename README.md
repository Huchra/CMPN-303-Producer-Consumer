# CMPN-303-Producer-Consumer

Producer Consumer Assignment for CMPN-303 course

## Instructions

1. Build  
   `make`

2. Run producer and consumer in separate terminals  
   `make produce p=<production rate> n=<buffer size>`  
   `make consume c=<consumption rate>`
